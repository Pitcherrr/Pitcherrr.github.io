The DFT and Circles

If you can rotate a line you can compute a DFT

\[\text{Say we want to find the DFT of the following signal:}\] \begin{flalign*} x[n] &= [1, 0, -1, 0] \end{flalign*} \[\text{The DFT is defined as:}\] \begin{flalign*} X_k &= \sum_{n=0}^{N-1} x[n] e^{\frac{-2 \pi j}{N}k\,n}\\ \text{For}\, X_{0\rightarrow N},\,\, e^{\frac{-2 \pi j}{N}k\,n} &= [e^{\frac{-2 \pi j}{4}\times0\times n}, e^{\frac{-2 \pi j}{4}n}, e^{\frac{-2 \pi j}{4}2n}, e^{\frac{-2 \pi j}{4}3n}]\\ &= [1, e^{\frac{-\pi j}{2}n}, e^{-\pi j\,n}, e^{\frac{-3 \pi j}{2}n}] \end{flalign*} \[\text{We can change these values to relations around the unit circle, note this is in degrees:}\] \begin{flalign*} &= [0^o, (-90\times n) ^o, (-180\times n) ^o, (-270 \times n) ^o]\\ \end{flalign*} \[\text{Now we treat pulses of x[n] as phasors around a circle:}\] \begin{flalign*} x[n] &= [1\angle 0^o, 0, 1 \angle 180^o, 0] \end{flalign*} \[\text{It is now much easier to complete the DFT as you are just rotating phasors around a circle:}\] \[\text{Solving the DFT:}\] \begin{flalign*} X_0 &= (1\angle 0^o \times \angle0^o) + 0 + (1 \angle180^o \times \angle0^o) + 0 = 0\\ X_1 &= (1\angle 0^o \times \angle0^o) + 0 + (1 \angle180^o \times \angle-180^o) + 0 = 2\\ X_2 &= (1\angle 0^o \times \angle0^o) + 0 + (1 \angle180^o \times \angle-360^o) + 0 = 0\\ X_3 &= (1\angle 0^o \times \angle0^o) + 0 + (1 \angle180^o \times \angle-540^o) + 0 = 2\\ \text{So we have, } X &= [0, 2, 0, 2] \end{flalign*}
